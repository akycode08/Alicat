# Как синхронизировать историю Cursor между macOS и Windows

## Почему ChatGPT синхронизируется, а Cursor нет?

**ChatGPT** - это облачный сервис. История хранится на серверах OpenAI и автоматически синхронизируется между всеми устройствами через ваш аккаунт.

**Cursor** - это локальное приложение. История чата хранится **локально на каждом компьютере** в файловой системе. Поэтому автоматической синхронизации нет.

---

## Инструкция для Windows

### Шаг 1: Найдите папку с историей Cursor на macOS

Откройте Terminal на Mac и выполните:

```bash
open ~/Library/Application\ Support/Cursor
```

Или вручную перейдите:
```
/Users/ваше_имя/Library/Application Support/Cursor/
```

### Шаг 2: Найдите файлы истории

История обычно хранится в одной из этих папок:

1. **User/globalStorage/** - общая история
2. **User/workspaceStorage/** - история по проектам
3. **User/History/** - если есть

Ищите файлы с названиями типа:
- `chat-history.json`
- `conversations.json`
- `*.db` (база данных)
- Папки с названиями проектов

### Шаг 3: Скопируйте файлы на Windows

**Вариант A: Через облако (Dropbox/OneDrive/Google Drive)**

1. Скопируйте папку `Cursor` из `~/Library/Application Support/` в облачную папку
2. На Windows: скопируйте из облака в `%APPDATA%\Cursor\`

**Вариант B: Через USB/сеть**

1. Скопируйте папку `Cursor` на флешку или сетевой диск
2. На Windows: вставьте в `%APPDATA%\Cursor\`

### Шаг 4: На Windows - путь к папке Cursor

Откройте Проводник Windows и в адресной строке введите:

```
%APPDATA%\Cursor
```

Или перейдите вручную:
```
C:\Users\ваше_имя\AppData\Roaming\Cursor\
```

### Шаг 5: Вставьте файлы истории

1. Закройте Cursor на Windows (если открыт)
2. Скопируйте файлы истории из Mac в соответствующую папку на Windows
3. Запустите Cursor на Windows
4. История должна появиться

---

## Альтернативный способ: Экспорт/Импорт

Если прямого копирования не работает:

1. **На Mac**: Найдите файлы истории и скопируйте их содержимое
2. **На Windows**: Создайте те же файлы вручную и вставьте содержимое

---

## Важно!

⚠️ **Перед копированием закройте Cursor на обоих компьютерах**, чтобы избежать конфликтов файлов.

⚠️ **Сделайте резервную копию** существующих файлов на Windows перед заменой.

---

## Если не работает

1. Проверьте права доступа к файлам
2. Убедитесь, что пути правильные
3. Перезапустите Cursor после копирования
4. Проверьте версии Cursor - они должны быть одинаковыми или совместимыми

---

## Быстрый путь для копирования

**На Mac:**
```
~/Library/Application Support/Cursor/User/globalStorage/
```

**На Windows:**
```
%APPDATA%\Cursor\User\globalStorage\
```

Скопируйте содержимое из Mac в Windows.
